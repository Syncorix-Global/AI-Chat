# Security Policy

## Supported Versions
We support the most recent **minor** versions (N and N-1) of this library.

## Reporting a Vulnerability
- **Private, responsible disclosure only.** Email: **security@syncorixglobal.com**
- Please include: affected versions, environment, PoC or steps to reproduce, expected vs. actual behavior, and any mitigation you suggest.
- You will receive an acknowledgement within **3 business days**.
- We aim to provide a remediation or workaround within **14 days**, with coordinated disclosure as needed.
